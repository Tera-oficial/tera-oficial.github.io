Un lenguaje de precisión

GrapeSum nace de la elegancia de la matemática pura. Un lenguaje nuevo, profesional, limpio y diseñado para la velocidad y claridad.

Ejemplo de sintaxis:

block.main:
  let n = 10;
  sum = Σ(i = 1; i <= n; i + 1) { i };
  print(sum);

Diseñado para desarrolladores exigentes

GrapeSum compila a C nativo, integrando lo mejor del rendimiento con una sintaxis lógica y expresiva. Totalmente compatible con contenedores Python y JavaScript.
